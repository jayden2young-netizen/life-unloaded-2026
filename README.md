# 人生尚未加载 · 2026

一款中文移动端人生模拟游戏。

出生、读书、工作、迁移、婚恋、疾病和养老会沿着时间线逐年发生。大多数年份只是生活继续往前，真正遇到岔路时才需要选择。早年的一个随手决定，可能很多年后才回来找你。

[在线游玩](https://jayden2young-netizen.github.io/life-unloaded-2026/)

## 怎么玩

1. 随机生成性别、出生地和家庭画像。
2. 分配初始属性，再选一张先天牌。
3. 轻触时间线，每次推进一条事件。
4. 遇到卡牌或人生选择时，游戏会暂停等待操作。
5. 活到这一生结束，查看寿命、关键经历、所得与所失。

每局通常有 12—15 次选择。选择会改变工作、城市、关系、债务、欲望满足度和未来回响；结局只从本局真实形成的选择组合与最终状态中产生。

## 现在的版本

当前开发版本为 v4.0.1「因果人生」，主要内容包括：

- 每年一个主要事件；回响、秘密和危机后果优先占用可选的第二事件位。
- 出生地、家庭、六项属性、前三欲望与核心矛盾共同改变事件权重和结果。
- 工作使用学生、受雇、灵活就业、自雇、失业、退休和照护假状态机；伴侣与子女使用真实关系账本。
- 家庭秘密、选择回响、债务利息、失业年限、健康与压力会形成后续事件和有代价的恢复路线。
- 黑天鹅具有独立年龄、状态门槛和实际效果，每局最多两次。
- 结局由具体 `outcomeTags`、关键转折和最终状态组合而成，并引用本局真实选择、回响及所得所失。
- 社会图鉴按事件、结果标签和状态里程碑跨局解锁，不再依赖文案匹配。
- 大学毕业、求职、考公与子女成长使用明确状态和阶段链；无子女晚年使用独立的单身与伴侣事件池。
- 移动端优先，适配 iPhone、Galaxy S 系列及较小的手机视口。

游戏使用克制的黑色幽默处理就业、住房、婚育、照护、AI、平台劳动和晚年数字生活。它会写到失业、疾病、关系破裂和死亡，但不包含露骨描写。

## 内容规模

- 600 个事件节点
  - 400 个年度事件
  - 100 个选择事件
  - 80 个长期回响
  - 20 个低概率黑天鹅
- 72 张卡牌
- 30 种家庭画像
- 44 个家庭秘密
- 16 条结局轨迹、64 个结局标题和 96 条条件片段
- 30 个跨局社会图鉴条目

游戏没有外部字体、图片、CDN 或第三方接口请求。

## 技术结构

项目是直接部署到 GitHub Pages 的静态网页，没有框架、npm 构建步骤和运行时依赖。

```text
index.html   页面入口与启动错误提示
style.css    黑色移动端界面与响应式布局
game.js      状态、随机数、存档和游戏流程
data.json    家庭、卡牌、事件与结局数据
```

`tools/` 用于生成和维护数据，`tests/` 保存静态校验与浏览器检查脚本，它们不会参与线上运行。

## 本地运行

由于游戏通过 `fetch()` 读取 `data.json`，不要直接双击 `index.html`。请在仓库目录启动一个静态文件服务器：

```bash
python -m http.server 8765
```

然后打开：

```text
http://127.0.0.1:8765/
```

页面和数据加载失败时会显示可读错误，不会停在黑屏。

## 存档

游戏使用 `localStorage` 自动保存当前人生、历史档案和跨局统计，不需要账号或服务器。

- 刷新或关闭页面后可以继续。
- v4.0 首次加载会备份完整的 v3.2.4 存档，保留历史、图鉴、设置与统计，并清除未完成的人生要求重新开局。
- 存档键继续使用 `life-unloaded-2026-v1`；当前 `schemaVersion` 为 5。
- 清除浏览器站点数据会同时删除本地存档。
- 设置页可以导出当前存档。

## 调试与校验

在地址后添加 `?debug=1` 可以打开开发调试入口：

```text
http://127.0.0.1:8765/?debug=1
```

静态数据校验：

```bash
node tests/v3-validate.mjs
```

移动端交互可达性检查需要 Playwright：

```bash
node tests/v3-responsive-smoke.cjs http://127.0.0.1:8765/
```

定向核心路径检查：

```bash
node tests/v3-browser-smoke.cjs http://127.0.0.1:8765/
```

这个项目仍在根据真实试玩继续调整。数值平衡、文案感受和重开欲望以人工体验为准，自动测试只负责发现加载、状态和布局错误。

# 人生尚未加载 · 2026

一款中文移动端因果人生模拟游戏。

你在 2026 年出生。出身决定最初能看见的门，教育、工作、不工作、迁移、经营、婚恋、子女、债务、健康、习惯与养老会沿着真实状态继续发展。选择不会保证成功，但会改变以后还剩哪些选择；过去的决定也会带着具体后果跨年返回。

[在线游玩](https://jayden2young-netizen.github.io/life-unloaded-2026/)

## 怎么玩

1. 随机生成性别、出生地、家庭成员和家庭账面。
2. 分配初始属性，再选一张会改变能力或风险的卡牌。
3. 轻触时间线，每次推进一条可见事件。
4. 青春期认领核心欲望，并在人生岔路中选择可承担的路线。
5. 在这一生结束后查看六维人生画像、三个真实转折和基于事实生成的结局。

一局通常约有 85—110 条时间线和 16—20 次关键选择。每年最多出现两条可见事件；到期义务、长期后果和连续故事优先。

## 当前版本

当前开发版本为 v5.0.3「健康与经济因果修复」。

- 使用唯一人生账本记录世界、原生家庭、教育、角色、就业、活动、个人财务、人物关系、健康、习惯、欲望、义务、连续故事和决定历史。
- 父母住房、资产和债务属于原生家庭，不再自动算作婴儿的个人财产。
- 教育、工作、求职、主动不工作、照护、退休和疾病退出分别保留真实状态；无工作者不会收到裁员、晋升或排班事件。
- 父母、手足、伴侣、前伴侣和子女是有年龄、存亡、关系及法律身份的真实人物实体。事件绑定不到对应人物时不会触发。
- 工资、生活成本、住房、医疗、经营、股权和逐笔债务进入同一年度结算。主动休息必须有资金来源，严重负债会形成逾期和后续责任。
- 成瘾与恢复采用接触、反复、依赖、失控、治疗、恢复和复发阶段，不用单个标签替代过程。
- 十二组人生轨道都覆盖进入、发展、日常、冲突、危机、恢复或退出和晚年回响；同时最多维持三条跨领域连续路线。
- 每个关键选择都保存选择前状态、具体选项、即时影响、承诺、跨年后果与结局贡献，结局不再按“遇到过某个主题”抽取。
- 结局取消综合分，展示自主、关系、健康、安全、欲望兑现和社会影响六维画像，以及三个真实转折、稀有等级、路线标签与种子。
- 保留黑色移动端 UI、单击推进、不可跳过底部弹层、种子随机、离线存档、跨局图鉴和四文件静态运行。

成人写实内容会涉及失业、疾病、关系破裂、成瘾、违法、债务和死亡，但表达保持克制，不以猎奇作为奖励。

## 内容规模

- 620 个事件节点
  - 400 个年度事件
  - 100 个选择事件
  - 100 个选择特定长期后果
  - 20 个低概率黑天鹅
- 72 张能力与风险卡牌
- 30 种家庭画像
- 44 个家庭秘密
- 12 组完整人生轨道
- 16 条结局画像、64 个结局标题
- 30 个跨局社会图鉴条目

v5 将长期后果从 80 条扩展到 100 条，使每个关键选择都拥有对应的选择特定后果；其余内容没有为了凑数量继续扩张。

游戏没有外部字体、图片、CDN、第三方接口或运行时依赖。

## 技术结构

项目直接部署到 GitHub Pages，没有框架或 npm 构建步骤。

```text
index.html   页面入口与启动错误提示
style.css    黑色移动端界面与响应式布局
game.js      状态账本、因果引擎、存档和游戏流程
data.json    由生成器产生的全部内容数据
```

`tools/generate-v5-data.mjs` 是 v5 数据源；`tools/generate-v3-data.mjs` 保留为兼容入口并转发到 v5 生成器。禁止只手工修改 `data.json`。

## 本地运行

由于游戏通过 `fetch()` 读取 `data.json`，不要直接双击 `index.html`。在仓库目录运行：

```bash
python -m http.server 8765
```

然后打开 `http://127.0.0.1:8765/`。调试模式为 `http://127.0.0.1:8765/?debug=1`，正式地址不会暴露调试接口。

## 存档与迁移

- 存档键继续使用 `life-unloaded-2026-v1`，当前 `schemaVersion` 为 7。
- 游戏版本变化时不再备份或迁移旧版本的活动人生，玩家会从新人生开始。
- 人生档案、图鉴、设置、统计、跨局新鲜度和种子记录会继续保留。
- 设置页可以导出存档，也可以用“清除全部数据”删除当前人生、跨局记录和遗留的旧版本数据。

## 校验

```bash
node --check game.js
node --check tools/generate-v5-data.mjs
node tests/v3-validate.mjs
node tests/v5-state-contract.mjs
node tests/v5-browser-smoke.cjs
```

浏览器脚本需要本地服务器和 Playwright。它只跑一次定向核心路径，验证 schema 迁移、选择特定后果、关键状态持久化、六维结局、零控制台错误，以及 360×773、360×640、320×568 三个移动端视口。

数值平衡、文案感受、危机强度和重开欲望继续由人工试玩判断；自动测试不做批量人生模拟。

# 人生尚未加载 · 2026

一款中文移动端人生模拟游戏。

你在 2026 年出生。出生地和家庭会影响最初的生活，之后读什么书、做什么工作、跟谁生活、要不要孩子、如何花钱，都由这一生里真正发生过的事继续往下推。

[在线游玩](https://jayden2young-netizen.github.io/life-unloaded-2026/)

## 怎么玩

1. 随机生成出生地、家庭成员和家庭情况。
2. 分配六项初始属性，再选一张伴随人生的卡牌。
3. 轻触时间线，每次推进一件事。
4. 遇到岔路时，从当下确实做得到的选项里选择。
5. 这一生结束后，查看人生画像、三次重要转折和专属结局。

一局通常有 85—110 条时间线和 16—20 次重要选择。游戏会自动存档，不需要注册账号。

## 你可能经历什么

有人一路读到大学，也有人读职校、考证、辍学后再回到课堂。工作可以是企业、公职、平台劳动、远程接单或自己开店；你也可以暂时不工作，把时间用来照护家人、养病或者单纯生活。

感情可能走向恋爱、同居、婚姻、分开和再婚。你可以生育、收养、成为继父母，也可以主动不要孩子。房子、债务、疾病、成瘾、养老和遗嘱不会只出现一次，它们可能过几年又回来，换一种方式影响你。

大多数人生不会整齐地成功或失败。普通日子、偶然好运和麻烦事会混在一起。极少数路线可以走到巨额财富，也可能留下还不完的债。

## 选择会留下什么

选择不会保证一个好结果。它可能带来收入，也可能占掉时间；保住一段关系，也可能让另一件事被耽误。家庭、能力、现金、健康和已经认识的人会决定你当时能做什么。

过去的选择会在几年后产生后果。结局只会引用这一生真实发生过的转折，不会因为碰巧见过某个主题就给出不相干的标题。

## 当前版本

当前版本是 v0.5.8，项目仍在早期开发中。

这一版把恋爱婚姻、成为父母与收养、子女成长边界，以及远程工作和旅居拆成独立短期事件簇。关系建立、婚姻危机、离婚、复合与晚年相伴不再被强行串成一条路；入学、青春期和成年边界按孩子实际年龄开启；远程合同、平台依赖、海外许可和固定基地也分别处理。长期关系、育儿和迁移仍作为生活状态继续。家庭秘密和部分结局文字还会继续修改。

目前共有：

- 400 个年度事件
- 124 个选择事件
- 124 个选择对应的长期后果
- 20 个低概率意外
- 72 张卡牌
- 30 种家庭
- 44 个家庭秘密
- 12 条人生路线
- 64 个结局标题
- 30 个跨局图鉴条目

游戏会涉及失业、疾病、关系破裂、成瘾、违法、债务和死亡。文字尽量克制，不会把痛苦写成猎奇奖励。

## 存档

存档保存在当前浏览器中。设置页可以导出存档，也可以使用“清除全部数据”删除当前人生和所有跨局记录。

版本更新后，尚未结束的一生可能需要重新开始；已经留下的人生记录、图鉴、设置和统计会继续保留。清理浏览器数据或更换设备前，建议先导出存档。

## 本地试玩

下载项目后，在项目目录启动一个本地服务器：

```bash
python -m http.server 8765
```

然后打开 `http://127.0.0.1:8765/`。游戏通过浏览器读取本地数据，直接双击 `index.html` 无法正常启动。

发现事件与年龄、工作、伴侣或子女状态对不上时，可以在 [GitHub Issues](https://github.com/jayden2young-netizen/life-unloaded-2026/issues) 留下当时的年龄、状态和事件文字。能附上种子编号会更容易复现。
